Behind The Map

We use Javalin, a Java web framework, to serve the frontend and the API endpoints. Route calculation is done by a custom implementation of the Dijkstra algorithm. Data is provided by the Institute for Formal Methods in Computer Science at the University of Stuttgart. The calculated routes are displayed using GeoJSON and the Leaflet JavaScript library.
